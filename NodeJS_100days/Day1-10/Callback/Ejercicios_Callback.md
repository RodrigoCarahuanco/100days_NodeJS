# Ejercicios Callbacks

1. Suma de dos números:
- Escribe una función llamada sumar que acepte dos números como parámetros y un callback. El callback debe ser invocado con la suma de los dos números como argumento. (repasar)

2. Filtrar elementos:
- Escribe una función llamada filtrar que acepte un array de elementos y una función de callback. La función debe filtrar los elementos del array según el resultado de la función de callback y devolver un nuevo array con los elementos filtrados. (repasar)

3. Ordenar array:
- Escribe una función llamada ordenar que acepte un array de números y una función de callback. La función debe ordenar los números en el array de acuerdo al resultado de la función de callback y devolver el array ordenado. ()

4. Validación de usuario:
- Escribe una función llamada validarUsuario que acepte un nombre de usuario, una contraseña y una función de callback. La función debe validar las credenciales del usuario según el resultado de la función de callback y devolver un mensaje de éxito o error.

5. Cálculo de promedio:
- Escribe una función llamada calcularPromedio que acepte un array de números y una función de callback. La función debe calcular el promedio de los números en el array y pasar el resultado a la función de callback.
